# Prompt 设计与工程

目标：为调用大模型/AI 组件提供稳定、可控的指令。

要点：
- 明确角色与限制：在 prompt 开头指定 assistant 角色、输出格式和禁止操作。
- 示例示范（Few-shot）：提供 1-3 个示例以限定期望风格与结构。
- 控制采样：根据需要设置 temperature、top_p、max_tokens 等参数。
- 结构化输出：要求返回 JSON 或表格以便程序化解析。
- 最小且明确的上下文：只提供必要的上下文，避免输入过多噪声。

示例：
- 系统："你是量化策略助理，输出必须是 JSON: {symbol, score, reason}" 
- 用户示例："给出前 10 名股票，按照 score 降序排列。"
