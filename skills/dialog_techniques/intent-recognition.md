# 意图识别（Intent Recognition）

目的：从用户输入中识别用户想要做什么（意图），并提取必要的槽位（slot）。

要点：
- 意图分类与置信度：使用分类模型或规则优先级，返回意图标签和置信度阈值（例如 0.7）。
- 槽位抽取（Slot Filling）：对需要参数的意图提取实体（例如日期、股票代码、策略名）。
- 多意图与优先级：允许多意图返回并按优先级排序；短语中包含多个意图时做拆分或询问确认。
- 误判与澄清策略：当置信度低于阈值或存在冲突，触发澄清问题（简洁、引导式）。
- 上下文依赖：利用会话上下文关联意图（例如前文已询问具体日期，则补全缺省日期）。

示例：
- 用户："帮我跑一下方案A今天的选股" → 意图: run_strategy, slots: {strategy: "A", date: today}
- 用户："要不要买腾讯" → 意图: question_advice, slots: {symbol: "0700.HK"}（若置信度低，反问确认）
