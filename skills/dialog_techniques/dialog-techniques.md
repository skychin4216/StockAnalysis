# 对话技巧速览

目标：提高助手的可用性、可控性与用户信任。

核心技巧：
- 简洁确认（Confirm）：对关键操作用一句话确认并给出默认选项。
- 分段引导（Chunking）：把复杂任务拆成小步并引导用户逐步授权。
- 主动建议与备选：给出建议同时列出 1-2 个备选方案，帮助决策。
- 回顾与总结：在完成关键步骤时给出简短摘要（例如已执行的策略与影响）。
- 透明性与可解释：在提供建议时给出理由（简短），并允许查看详细来源或参数。
- 错误回退（Fallback）：无法理解时先提供重述选项，再提供人工/更多示例。

示例用法：
- "我可以帮你先运行量化选股（方案A与B），要按星级优先还是按评分合并？"
- "我不确定你指的是今天还是最近一周，想看今天的结果吗？(是/否)"
