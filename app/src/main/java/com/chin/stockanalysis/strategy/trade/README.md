# 模拟交易系统架构文档

## 概述
模拟交易系统是 StockAnalysis v8.0 的核心新增功能，提供完整的模拟买卖、回溯验证、调优拟合和数据导入导出能力。

## 核心数据表

### 1. daily_period_result（策略周期结果）
记录每个策略在每个周期的执行结果：
- **原始信号**: 策略输出的5-10只原始股票
- **新闻力度评分**: 根据新闻因子的影响强度和情感分计算
- **板块轮动惩罚**: 检测到板块连续上涨≥3天时给予惩罚
- **主板过滤**: 记录被过滤的股票及原因（创业板/科创板/北交所）
- **最终Top3**: 经过新闻加权和轮动调整后的前三名
- **AI精选理由**: 跨策略聚合后的AI选择理由

### 2. strategy_trade_fitting_params（调优拟合参数）
记录每个策略在每个周期的1000轮调优参数：
- **param_json**: 每轮扰动后的权重参数JSON
- **fitting_round**: 第N轮调优
- **accuracy**: 该轮调优在次日验证中的准确率
- **avg_return**: 该轮调优在次日验证中的平均收益率
- **查询方式**: 按策略+日期+周期查询，可找出最佳/最差参数

### 3. daily_news_hot_picks（每日新闻热点固化）
固化每天新闻热点的Top3：
- **聚合规则**: 按板块聚合新闻因子的影响强度评分
- **关联股票**: 记录该板块下有哪些相关股票
- **用途**: 收盘后可对比「当天新闻热点」与「股票最热前三名」是否接近

### 4. strategy_trade_orders（模拟交易订单）
完整记录每笔模拟交易：
- **状态流转**: PENDING → BUYING → HELD → SOLD/FAILED
- **买入信息**: 日期、价格、数量、买入理由、评分
- **卖出信息**: 日期、价格、收益百分比
- **回溯**: 次日点击「回溯」按钮自动查次日收盘价更新状态

## 板块轮动规则

### 连续上涨检测
- 阈值: 板块连续上涨 ≥ 3天
- 惩罚计算: `penalty = -(count - 3 + 1) * 10`
- 范围: -100 ~ 0

### 新闻加权
- 从 news_factors 表获取最近3天的活跃新闻
- 计算与信号股票相关的新闻平均强度分 + 情感分
- 加权幅度: `strength * (1.0 + newsBoost + rotationAdj)`
- newsBoost = (newsScore - 50) / 100 * 0.15
- rotationAdj = rotationPenalty / 100 * 0.10

## AI精选算法
```
综合评分 = 平均强度 × 0.6 + 被选中策略数占比 × 0.4
```
- 跨所有策略的所有周期聚合
- 最终选出3只综合评分最高的股票
- 上涨概率 = avgScore × 0.8 + 20

## 热门板块子板块展开规则

### 展开逻辑
1. 取概念板块**前三名**
2. 每个板块通过 SectorSubDivision 查询子板块
3. **不限制子板块数量**，全部列出
4. 如：光通信 → [光芯片、光模块、CPO、光材料、光纤...]
5. 没有子板块的板块保留原名

### 子板块映射（硬编码）
部分通过 SectorSubDivision 获取，部分通过 `StrategyListFragment.hardcodedSubSector()` 硬编码的股票名称→子板块映射，例如：
- 三环/风华/火炬/洁美 → MLCC
- 中际/新易盛/天孚/光迅 → 光模块
- 长飞/亨通/中天 → 光纤

## 数据导出导入

### JSON格式
- 包含10张表的完整数据
- 文件命名: `stock_analysis_YYYY-MM-DD.json`
- 可在手机间互传、备份

### CSV格式
- 分表导出，每张表一个CSV文件
- 可直接用Excel打开分析
- 文件命名: `{table_name}_YYYY-MM-DD.csv`

### 文件路径
`内部存储/Android/data/com.chin.stockanalysis/files/StockAnalysis_exports/`

## 使用流程

### 首次使用
1. 打开App → 策略Tab → 量化选股
2. 点击「导入」拉取历史K线数据
3. 等待数据完成

### 每日操作
1. 量化选股 → 执行策略 → 查看扫描结果
2. 模拟交易Tab → 选择交易日/周期
3. 执行模拟交易 → AI精选Top3 → 自动买入
4. 次日 → 点击「回溯」查看收益
5. 定期点击「拟合」查看调优效果

### 数据迁移
1. 点击「数据」→ 导出JSON
2. 通过USB复制文件到新手机
3. 新手机上点击「数据」→ 导入 → 选择JSON文件
